Claude Skill

あなたの代わりに、
GitHubを操作するスキル。

リポジトリの作成からファイルのpush、PR、Issues、GitHub Actions、Pages公開まで。Claudeに一言伝えるだけで、GitHub上のあらゆる作業を任せられます。

clidhub — session

$ claude "新しいリポジトリを作って、READMEとCIを設定して公開して"

→ Clidhub skill triggered

✓ gh repo create my-project --public

✓ git push -u origin main

✓ .github/workflows/ci.yml committed

✓ GitHub Pages enabled (/docs)

https://github.com/you/my-project

$

capabilities

リポジトリ操作
作成・削除・clone・可視性変更・コラボレーター招待をghとREST APIで実行。
ファイル構築
複数ファイルのコミット&push、単一ファイルの軽量編集をワークフローに応じて自動選択。
ブランチ / PR
ブランチ作成、プルリクエストの作成・マージ・クローズまで一連の流れを自動化。
Issues
Issueの作成・編集・ラベル付け・クローズをコマンド一発で。
GitHub Actions
CI/CD workflowをテンプレートから生成・push。実行状況もgh runで確認。
GitHub Pages
静的サイトの公開設定をREST API経由で有効化。公開URLまで提示。
Projects (v2)
プロジェクトボードの作成・Issue追加・フィールド更新をgh projectとGraphQLで実行。
プロフィール読み取り
プロフィール・フォロワー・メールアドレス・プランを読み取り専用で取得。

setup

requirements

1. GitHub Personal Access Token を発行する(repo / workflow スコープ、または Fine-grained PAT で Contents・Pull requests・Issues・Actions・Pages の Read and write)

2. .skill ファイルを Claude にインストールする

3. セッションごとにトークンを聞かれたら貼り付ける(永続保存はされません)

install clidhub

.skill ファイルをダウンロードして、Claudeにインストールしてください。
